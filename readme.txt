Introduction
កម្មវិធីនេះត្រូវបានអភិវឌ្ឍឡើងដោយប្រើភាសា C និង C++ ដើម្បីបង្ហាញពីការដំណើរការនៃ Algorithm ស្វែងរកទិន្នន័យ ដូចជា Linear Search និង Binary Search ក្នុងបណ្ដុំទិន្នន័យ (Array / List)។ កម្មវិធីនេះត្រូវបានបង្កើតឡើងសម្រាប់ជួយដល់សិស្សានុសិស្ស និងអ្នកចាប់អារម្មណ៍ក្នុងវិស័យ IT ឲ្យយល់ដឹងពីរបៀបដំណើរការ និងប្រៀបធៀបប្រសិទ្ធភាពរបស់ Algorithm នីមួយៗ។ វាជាជំហ៊ានដំបូងក្នុងការបង្កើតប្រព័ន្ធឆ្លាតវៃ និងកម្មវិធីមានប្រយោជន៍សម្រាប់សហគ្រាសនានាក្នុងសេដ្ឋកិច្ចសម័យថ្មី។

Objective
កម្មវិធីនេះបង្ហាញពីការដំណើរការ Linear និង Binary Search ដោយប្រើភាសា C/C++។ វាជួយសិស្ស និងអ្នកអភិវឌ្ឍយល់ Algorithm និងប្រៀបធៀបប្រសិទ្ធភាព សម្រាប់អនុវត្តក្នុងប្រព័ន្ធឆ្លាតវៃនាពេលអនាគត។

Methodology
1. Linear Search
- អភិវឌ្ឍកម្មវិធីស្វែងរកទិន្នន័យដោយធៀប Element ម្ដងមួយៗ ក្នុងបណ្ដុំ (Array)
- ប្រើលក្ខខណ្ឌ if ដើម្បីផ្ទៀងផ្ទាត់តម្លៃដែលត្រូវការ
- ត្រូវបានអនុវត្តក្នុងភាសា C/C++
2. Binary Search
- ស្រង់ Array នឹងតម្រៀបសិន​ (Sort Array)
- ប្រើវិធីបែងចែកខ្ទង់ (Divide and Conquer) ដើម្បីស្វែងរកតម្លៃ
- អនុវត្តដោយប្រើលក្ខខណ្ឌ while ឬ recursion ក្នុងភាសា C/C++

Conclusion
- Linear Search គឺជាវិធីសាស្ត្រស្វែងរកមួយដែលធ្វើការត្រួតពិនិត្យធាតុក្នុងបណ្ដុំមួយនៃ Array ម្ដងមួយៗ មករហូតដល់ឃើញតម្លៃដែលចង់បាន ឬដល់ចុងបណ្ដុំ (Array)។ វាអាចដំណើរការបានទាំងលើទិន្នន័យដែលតម្រៀបរួច និងមិនទាន់តម្រៀប។ វាជាវិធីសាមញ្ញដែលងាយស្រួលក្នុងការអនុវត្តតែនៅមានពេលវេលាចំណាយខ្ពស់ក្នុងករណីទិន្នន័យច្រើន។
- Binary Search ជាវិធីសាស្ត្រស្វែងរកដែលមានប្រសិទ្ធភាពខ្ពស់សម្រាប់ទិន្នន័យដែលបានតម្រៀបរួច (Array sorted)។ វាធ្វើការបែងចែកដងស្វែងរកជាបីភាគជាប់ៗគ្នា ដោយប្រៀបធៀបតម្លៃគោលដៅជាមួយតម្លៃកណ្តាលរបស់បណ្ដុំ ហើយបន្តស្វែងរកតែផ្នែកដែលអាចមានតម្លៃនោះ។ វាធ្វើឱ្យការស្វែងរកលឿន និងមានប្រសិទ្ធភាពក្នុងករណីទិន្នន័យច្រើន។
